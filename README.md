## 🐛 Issues & Solutions

### 1. Can't find the Extensions menu in Cursor

**Issue:** Hard to find the Extensions menu when first using Cursor IDE — the interface wasn't familiar.

**Solution:** Cursor has two different views — the **Agent window** and the **Editor window**. The Extensions menu is only available in the **Editor window**. Switch to it to access extensions.
